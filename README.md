## ▶ &nbsp; Start Streaming — Local Setup

```bash
# 1. Clone the repository
git clone https://github.com/your-username/portfolio-v2.git
cd portfolio-v2

# 2. Install dependencies
npm install

# 3. Configure environment variables
cp .env.example .env.local
# Fill in your Supabase, Twilio, and reCAPTCHA credentials

# 4. Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — *choose your profile and start browsing.*

<br/>

### Environment Variables

| Variable | Description |
|----------|-------------|
| `NEXT_PUBLIC_SUPABASE_URL` | Your Supabase project URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase anonymous key |
| `SUPABASE_SERVICE_ROLE_KEY` | Supabase service role key |
| `TWILIO_ACCOUNT_SID` | Twilio account SID (contact form) |
| `NEXT_PUBLIC_RECAPTCHA_SITE_KEY` | Google reCAPTCHA site key |
